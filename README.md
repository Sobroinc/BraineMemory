# BraineMemory v2.2

Universal memory system for AI agents via MCP.

## Features

- **Unified storage**: Documents, graphs, vectors in SurrealDB
- **MCP protocol**: Standard interface for AI tools
- **Embeddings**: text-embedding-3-large (dim=3072)
- **Vision**: gpt-5.2 for image/document analysis

## Quick Start

```bash
pip install -e .
python -m src.mcp.server
```
